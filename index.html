Object fit

La propiedad CSS de object-fit establece cómo se debe cambiar el tamaño del contenido de un elemento reemplazado, como img o video, para que se ajuste a su contenedor. Puede alterar la alineación del objeto de contenido del elemento reemplazado dentro del cuadro del elemento utilizando la propiedad de object-position del objeto.

Ejemplo:

        
        object-fit: contain;
        object-fit: cover;
        object-fit: fill;
        object-fit: none;   
        object-fit: scale-down;

        / * Valores globales * /
        object-fit: inherit;
        object-fit: initial;
        object-fit: revert;
        object-fit: revert-layer;
        object-fit: unset;

Responsive Design

El principio de responsive design se logra a través de hojas de estilos en cascada (CSS) y técnicas de programación (HTML). Con estas técnicas, puede evitar que se integre el mismo contenido varias veces, ya que el objetivo del responsive design es adaptar el mismo conjunto de contenidos a las diferentes tecnologías que se utilizan.

Mobile First

Como su propio nombre indica, esta metodología trata de dar prioridad al diseño de una web en versión móvil y, posteriormente, se adapta a PC.

Ejemplo:

        
        @media (max-width: 767px) {
            /* Estilos para móvil */
        }
